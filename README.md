1. REST API merupakan sebuah metode komunikasi antara klien dan server pada HTTP dengan menggunakan method -method http yang semantic (get,post put, dll) untuk mengelola resource dan biasanya memiliki output json atau xml

CORS (cross origin resource sharing) adalah mekanisme keamaanan berbasis header http untuk akses smber daya berdasarkan origin/domain nya, CORS umum dijumpai di browser. cara mengatasi nya denagan memberi http header Access-Control-Allow-Origin response untuk origin/domain yang dituju , framework-framwork  web biasanya sudah memiliki middleware untuk menangani CORS.

SQL merupakan database relasional, setiap data, dapat terhubung dengan data lainnya melalui relasi ini. schema SQL juga tetap dan terstruktur, SQL biasanya berbasis tabel contohnya PostgresSQL dan MySql. sedangakan. NOSQL tidak mewajibkan relasi dan skema nya fleksibel biasnya berbasis dokumen, key-value,column store. contohnya Redis dan Mongo DB
4. middleware adalah komponen perantara / gerbang lapis antara request dan handler response utama pada aplikasi web. fungsinya seperti memprosess, memeriksa, memodifikasi request.
